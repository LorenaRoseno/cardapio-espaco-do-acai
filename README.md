# Espaço do Açaí - Cardápio Digital

## Sobre o projeto

Este projeto foi desenvolvido como parte da Atividade Extensionista do curso de Engenharia de Software.

A proposta da atividade consistiu na criação de uma solução digital para apoiar um negócio real, permitindo que clientes realizem pedidos de forma prática por meio de um cardápio online integrado ao WhatsApp.

O sistema foi desenvolvido para o estabelecimento **Espaço do Açaí**, com foco na melhoria da experiência do cliente e na organização dos pedidos recebidos.

---

## Funcionalidades

* Visualização de produtos organizados por categorias
* Seleção de tamanhos
* Escolha de opções obrigatórias (frutas, bombons, sorvetes e coberturas)
* Inclusão de adicionais com controle de quantidade
* Sacola de compras com atualização automática de valores
* Cadastro de dados do cliente e endereço de entrega
* Escolha da forma de pagamento
* Controle de troco para pagamentos em dinheiro
* Geração automática da comanda
* Envio do pedido diretamente para o WhatsApp
* Interface responsiva para dispositivos móveis

---

## Demonstração

O sistema permite que o cliente:

1. Navegue pelas categorias de produtos.
2. Personalize seu pedido com opções e adicionais.
3. Adicione itens à sacola.
4. Informe dados para entrega.
5. Escolha a forma de pagamento.
6. Envie automaticamente a comanda completa para o WhatsApp do estabelecimento.
